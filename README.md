#  Dashboard de Vendas - E-commerce

##  Visão Geral do Projeto

Este projeto tem como objetivo analisar dados de vendas de um e-commerce para extrair insights sobre desempenho de vendas, comportamento dos clientes e padrões de consumo.

A análise foi realizada utilizando Python (Pandas) e os resultados foram visualizados em um dashboard interativo no Power BI.

##  Principais Insights

* Clientes recorrentes são responsáveis pela maior parte do faturamento.
* O ticket médio se mantém relativamente estável na maioria das compras.
* Sessões mais longas tendem a gerar maior valor de compra, embora sejam menos frequentes.
* As categorias de produtos apresentam distribuição equilibrada de receita.
* O comportamento do cliente varia de acordo com o tempo de navegação no site.

##  Dataset

O dataset contém informações como:

* Pedidos e clientes
* Categorias de produtos
* Valores de compra e descontos
* Comportamento do usuário (tempo de sessão, páginas visualizadas)
* Tempo de entrega e avaliação do cliente

---

##  Conclusão

O projeto oferece uma visão clara sobre o desempenho de vendas e o comportamento dos clientes, permitindo identificar padrões importantes que podem auxiliar na tomada de decisões estratégicas.
